FFGPU Open Source Project

The FFGPU project implements:

  • An OpenGL Compliant Graphics Rendering Hardware.
  • A Programmable GLSL Compliant Shader SIMD/SIMT Processors.
  • A Full Software Stack of OpenGL Implementation on Linux.

rendering pipeline

A full OpenGL Compliant Graphics Rendering Pipeline

FFGPU implements the OpenGL-1.x and OpenGL-2.x Fixed function Graphics Rendering Pipeline and the new core-profile OpenGL-3x and OpenGL-4.x functions.

OpenGL shaders

All Programmable and GLSL Compliant

FFGPU is a all programmable hardware and support OpenGL GLSL compliant shaders.

  • Virtex Shader
  • Geometry Shader
  • Fragment Shader
  • Compute Shader

Mesa Software Driver

Based on Open Source Mesa3d

FFGPU Software Project implements Mesa3d's Gallium Driver. Just like AMD or Intel open source GPU driver.

Linux Display Driver

DRM/KMS Linux Kernel Driver

FFGPU Driver Project implements Linux's DRM/KMS display driver and support X11/Wayland display server.
